# InventoryBE

This project was created with [NodeJS] with version 10.0 

## Development server

Run `npm start` for a dev server. Use base URL as `http://localhost:8088/`. The app will automatically reload if you change any of the source files.

### API
`inv/login`
`inv/signUp`
`inv/getProducts`
`inv/getProductById`
`inv/addProduct`
